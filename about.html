Sobre mim

Meu nome é Lucas, comecei meus estudos na área de desenvolvimento aos 15 anos quando entrei no curso de informática da Escola Técnica Monteiro Lobato. Durante o curso tive contato com diversas linguagens de programação, como Java, C, JavaScript, MySQL, entre outras. E também alguns frameworks como Bootstrap, React, NodeJS, Express, Vuejs etc. Antes disso, estudava inglês desde os 12 anos, e hoje tenho um nível avançado.

Para o desenvolvimento do meu trabalho de conclusão fiz um site para postar opiniões musicais utilizando arquitetura MVC e banco de dados MySQL. O site foi desenvolvido utilizando EJS, CSS, JavaScript e MySQL. E consumindo a API do spotify para buscar as informações das músicas, API cloudinary para conseguir colocar fotos por link no banco de dados.
